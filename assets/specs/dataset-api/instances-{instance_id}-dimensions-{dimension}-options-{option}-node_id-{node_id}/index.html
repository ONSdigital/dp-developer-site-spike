/instances/{instance_id}/dimensions/{dimension}/options/{option}/node_id/{node_id}

PUT

Update the dimension with a nodeId

Parameters

instance_id - A unique id for an instance

dimension - A dimension from a dataset

node_id - An unique node id

option - A option to set within a type

Responses

400 - Failed to process the request due to invalid request
401 - The token provided is unauthorised to carry out this operation
403 - The request is forbidden
404 - InstanceId does not match any instances
500 - Failed to process the request due to an internal error
200 - Updated the dimension with the nodeId
