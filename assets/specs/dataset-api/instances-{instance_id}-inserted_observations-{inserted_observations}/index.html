/instances/{instance_id}/inserted_observations/{inserted_observations}

PUT

This will add to the number already store in the api

Parameters

instance_id - A unique id for an instance

inserted_observations - A value to increment the inserted_observations within an instance

Responses

401 - The token provided is unauthorised to carry out this operation
404 - InstanceId does not match any instances
500 - Failed to process the request due to an internal error
200 - Added value to inserted observation
400 - Failed to process the request due to invalid request
