/instances

GET

Get a list of instances which has been paged

Parameters

limit - The number of resources to be returned

offset - The first row of resources to retrieve, starting at 0. Use this parameter as a pagination mechanism along with the limit parameter

state - A comma separated list of state values to filter on (e.g. ‘completed,edition-confirmed’)

Responses

200 - Failed to process the request due to an internal error
400 - Failed to process the request due to an internal error
401 - Failed to process the request due to an internal error
500 - Failed to process the request due to an internal error

Create an instance which will be imported. To create an instance an import job id and href is required. This is to allow a link back to the import job

Parameters

instance - An instance related to an import job

Responses

500 - The token provided is unauthorised to carry out this operation
