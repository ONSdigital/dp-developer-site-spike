/datasets/{id}

GET

The dataset contains all high level information, for additional details see editions or versions of a dataset.

Parameters

id - Id that represents a dataset

Responses

200 - Failed to process the request due to an internal error
404 - Failed to process the request due to an internal error
500 - Failed to process the request due to an internal error

POST

Create a dataset provided by the ONS that can be filtered using the filter API

Parameters

id - Id that represents a dataset

dataset - A new dataset

Responses

500 - Forbidden to overwrite dataset, already published
201 - Forbidden to overwrite dataset, already published
400 - Forbidden to overwrite dataset, already published
401 - Forbidden to overwrite dataset, already published
403 - Forbidden to overwrite dataset, already published

PUT

Update the metadata for the next release of the dataset. The dataset contains all high level information, for additional details see editions or versions of a dataset.

Parameters

id - Id that represents a dataset

dataset - A new dataset

Responses

200 - Failed to process the request due to an internal error
400 - Failed to process the request due to an internal error
401 - Failed to process the request due to an internal error
404 - Failed to process the request due to an internal error
500 - Failed to process the request due to an internal error

DELETE

Delete an existing dataset

Parameters

id - Id that represents a dataset

Responses

500 - Forbidden to delete dataset, already published
204 - Forbidden to delete dataset, already published
401 - Forbidden to delete dataset, already published
403 - Forbidden to delete dataset, already published
