/datasets/{id}

GET

The dataset contains all high level information, for additional details see editions or versions of a dataset.

Parameters

id - Id that represents a dataset

Responses

500 - No dataset was found using the id provided
200 - No dataset was found using the id provided
404 - No dataset was found using the id provided

POST

Create a dataset provided by the ONS that can be filtered using the filter API

Parameters

id - Id that represents a dataset

dataset - A new dataset

Responses

403 - Unauthorised to create/overwrite dataset
500 - Unauthorised to create/overwrite dataset
201 - Unauthorised to create/overwrite dataset
400 - Unauthorised to create/overwrite dataset
401 - Unauthorised to create/overwrite dataset

PUT

Update the metadata for the next release of the dataset. The dataset contains all high level information, for additional details see editions or versions of a dataset.

Parameters

id - Id that represents a dataset

dataset - A new dataset

Responses

400 - A json object for a single Dataset
401 - A json object for a single Dataset
404 - A json object for a single Dataset
500 - A json object for a single Dataset
200 - A json object for a single Dataset

DELETE

Delete an existing dataset

Parameters

id - Id that represents a dataset

Responses

403 - Unauthorised to delete the dataset
500 - Unauthorised to delete the dataset
204 - Unauthorised to delete the dataset
401 - Unauthorised to delete the dataset
