/instances/{instance_id}/events

POST

Add a new event into an instance. Events can be for information or error messages. Each event must contain a type of event (Info or Error), a message to explain the event, time of the event and finally the message offset in kafka

Parameters

instance_id - A unique id for an instance

event - An event that occurs when importing a dataset

Responses

500 - InstanceId does not match any instances
201 - InstanceId does not match any instances
400 - InstanceId does not match any instances
401 - InstanceId does not match any instances
404 - InstanceId does not match any instances
