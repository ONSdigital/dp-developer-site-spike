/instances/{instance_id}

GET

Get the current state of an instance, this includes all events which have happened.

Parameters

instance_id - A unique id for an instance

Responses

200 - Failed to process the request due to an internal error
401 - Failed to process the request due to an internal error
404 - Failed to process the request due to an internal error
500 - Failed to process the request due to an internal error

PUT

Update an instance by providing an unique id and a set of properties to over write

Parameters

instance_id - A unique id for an instance

instance - An instance related to an import job and dataset

Responses

400 - The instance has been updated
401 - The instance has been updated
403 - The instance has been updated
404 - The instance has been updated
500 - The instance has been updated
200 - The instance has been updated
