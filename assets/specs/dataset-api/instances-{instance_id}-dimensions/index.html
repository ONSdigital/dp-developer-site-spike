/instances/{instance_id}/dimensions

GET

Get all dimensions from an instance

Parameters

instance_id - A unique id for an instance

Responses

200 - Return a list of dimensions
400 - Failed to process the request due to invalid request
401 - The token provided is unauthorised to carry out this operation
403 - The request is forbidden
404 - The instance was not found
500 - Failed to process the request due to an internal error

POST

Create a new dimension which is related to an instance

Parameters

instance_id - A unique id for an instance

dimension_option - A dimension option from an instance

Responses

404 - The instance was not found
500 - Failed to process the request due to an internal error
201 - Dimension was created
400 - Failed to process the request due to invalid request
