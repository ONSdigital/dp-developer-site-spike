/instances/{instance_id}/dimensions/{dimension}/options/{option}

PUT

Add a option to a dimension which has been found in the V4 data

Parameters

instance_id - A unique id for an instance

dimension - A dimension from a dataset

option - A option to set within a type

Responses

500 - InstanceId does not match any instances
200 - InstanceId does not match any instances
400 - InstanceId does not match any instances
401 - InstanceId does not match any instances
403 - InstanceId does not match any instances
404 - InstanceId does not match any instances
