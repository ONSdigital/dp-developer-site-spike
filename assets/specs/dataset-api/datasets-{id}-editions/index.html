/datasets/{id}/editions

GET

Get a list of editions from a type of dataset

Parameters

id - Id that represents a dataset

limit - The number of resources to be returned

offset - The first row of resources to retrieve, starting at 0. Use this parameter as a pagination mechanism along with the limit parameter

Responses

500 - No editions were found for the id provided
200 - No editions were found for the id provided
400 - No editions were found for the id provided
404 - No editions were found for the id provided
