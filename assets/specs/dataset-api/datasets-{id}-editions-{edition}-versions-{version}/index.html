/datasets/{id}/editions/{edition}/versions/{version}

GET

Get a specific version and edition of a dataset

Parameters

edition - An edition of a dataset

id - Id that represents a dataset

version - A version of a dataset

Responses

400 - Invalid request, reasons can be one of the following: * dataset id was incorrect * edition was incorrect
404 - No version was found for an edition of a dataset using the id, edition and version provided
500 - Failed to process the request due to an internal error
200 - A json object containing the edition and version of a dataset

PUT

Update a version for an edition of a dataset, if the state is changed to associated or published, the parent documents(dataset and edition resources) will also be updated. A version can only be updated if the state is not published

Parameters

id - Id that represents a dataset

edition - An edition of a dataset

version - A version of a dataset

Responses

403 - Forbidden to overwrite version of dataset, already published
404 - Version was not found for a dataset using the id and edition provided
500 - Failed to process the request due to an internal error
201 - A json object containing a version
400 - Invalid request, reasons can be one of the following: * invalid request body * dataset id was incorrect * edition was incorrect
401 - Unauthorised to update version of dataset
