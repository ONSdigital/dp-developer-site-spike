/datasets/{id}/editions/{edition}/versions/{version}

GET

Get a specific version and edition of a dataset

Parameters

edition - An edition of a dataset

id - Id that represents a dataset

version - A version of a dataset

Responses

200 - Failed to process the request due to an internal error
400 - Failed to process the request due to an internal error
404 - Failed to process the request due to an internal error
500 - Failed to process the request due to an internal error

PUT

Update a version for an edition of a dataset, if the state is changed to associated or published, the parent documents(dataset and edition resources) will also be updated. A version can only be updated if the state is not published

Parameters

id - Id that represents a dataset

edition - An edition of a dataset

version - A version of a dataset

Responses

201 - Failed to process the request due to an internal error
400 - Failed to process the request due to an internal error
401 - Failed to process the request due to an internal error
403 - Failed to process the request due to an internal error
404 - Failed to process the request due to an internal error
500 - Failed to process the request due to an internal error
