/instances/{instance_id}/dimensions/{dimension}/options

GET

Get all unique options from a dimension

Parameters

instance_id - A unique id for an instance

dimension - A dimension from a dataset

Responses

400 - Return a list of unique options
401 - Return a list of unique options
403 - Return a list of unique options
404 - Return a list of unique options
500 - Return a list of unique options
200 - Return a list of unique options
