Dataset API

An API used to query information about datasets published by the ONS.
Base path:
/v1
Version:
1.0.0

Paths

/datasets

GET

Returns a list of all datasets provided by the ONS that can be filtered using the filter API

limit query The number of resources to be returned false offset query The first row of resources to retrieve, starting at 0. Use this parameter as a pagination mechanism along with the limit parameter false

/datasets/{id}

GET

The dataset contains all high level information, for additional details see editions or versions of a dataset.

id path Id that represents a dataset true

/datasets/{id}/editions

GET

Get a list of editions from a type of dataset

id path Id that represents a dataset true limit query The number of resources to be returned false offset query The first row of resources to retrieve, starting at 0. Use this parameter as a pagination mechanism along with the limit parameter false

/datasets/{id}/editions/{edition}

GET

The edition contains a link to all versions

edition path An edition of a dataset true id path Id that represents a dataset true

/datasets/{id}/editions/{edition}/versions

GET

Get a list of all versions for an edition of a dataset

edition path An edition of a dataset true id path Id that represents a dataset true limit query The number of resources to be returned false offset query The first row of resources to retrieve, starting at 0. Use this parameter as a pagination mechanism along with the limit parameter false

/datasets/{id}/editions/{edition}/versions/{version}

edition path An edition of a dataset true id path Id that represents a dataset true version path A version of a dataset true

edition path An edition of a dataset true id path Id that represents a dataset true version path A version of a dataset true limit query The number of resources to be returned false offset query The first row of resources to retrieve, starting at 0. Use this parameter as a pagination mechanism along with the limit parameter false

/instances

GET

Get a list of instances which has been paged

Get the current state of an instance, this includes all events which have happened.
